# Premium CSS Monogram Logo Library

A static CSS-only logo/monogram reference library for boutique local business website builds.

## Web pages

- `index.html` — two-letter monogram set
- `index-1.html` — one-letter monogram set
- `index-3.html` — three-letter monogram set

Each page includes:

- 21 premium CSS-only logo cards
- matching dark-background test strip
- no JavaScript
- no image assets
- no external logo dependencies

## Design direction

Built for refined local business identities: salons, ateliers, spas, cafes, studios, and other boutique brands.

Favored styles:

- double-ring seals
- atelier ovals
- beveled medallions
- onyx cabochons
- embossed/debossed seals
- wax coins and stamps
- raised cameos
- champagne foil
- lacquer/enamel marks
- glass/marble/stone depth

Rejected directions intentionally omitted:

- diagonal anti-sign slashes
- orbit/science marks
- loading arcs
- crest/shield marks
- clover/medical petal marks
- novelty CodePen-style gimmicks

## Local usage

Open the HTML files directly in a browser or copy individual CSS/HTML mark patterns into client mockups.
